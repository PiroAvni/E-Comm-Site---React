# React E-Comms Site

A basic web site that includes features like:  
* Item display, 
* Item selection, 
* Cart - Add Items, Delete Items, 
* Checkout, 
* Authenication,
* Database Storage
* Payment system.

# Technology Stack Used
1. Reactjs
2. Redux ( State Management)
3. Javascript
4. Firebase (Authenication & Database storage)
5. Nodejs
6. Stripe (API) (Payment Service)
